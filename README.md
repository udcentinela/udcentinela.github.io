# UD Centinela - GitHub Pages

Esta carpeta es la version estatica publica de la web, preparada para GitHub Pages.

## Publicacion recomendada

1. Crear en GitHub un repositorio llamado `TU_USUARIO.github.io`.
2. Cambiar `TU_USUARIO` en los metadatos de esta carpeta por tu usuario real de GitHub.
3. Subir estos archivos a la rama `main`.
4. En GitHub, entrar en `Settings > Pages` y activar `Deploy from a branch`, rama `main`, carpeta `/root`.

La web quedara disponible en:

```text
https://TU_USUARIO.github.io/
```

## Importante

Esta version no incluye panel online para crear/editar noticias. Para GitHub Pages, las noticias se actualizan cambiando los archivos y volviendo a subir la web.

No se incluyen archivos privados como `auth.json`, `admin-password.txt`, backups ni scripts del panel admin.
